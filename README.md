# f-ck
F#ck is a small, expressive, functional‑first programming language inspired by F#, OCaml, and the broader ML family — but with a playful edge. It’s designed to be minimal, hackable, and fun to extend. 

I have no business creating my own programming language, but as I was starting to learn F# in Exercism, I wondered if anyone had created the obviously needed F# based language, F#ck and when I couldn't find it, well.. here we are.

Feel free to make notes on what should be changed - I'm still a beginner and probably have a lot that is wrong with the language.
